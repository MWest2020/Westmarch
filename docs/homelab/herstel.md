# Homelab

De homelab-documentatie (node-overzicht, netwerkmodel, herstelvolgorde,
GitOps-stappenplannen en terugkerende fixes) staat in de publieke import:
zie de sectie **homelab** in de navigatie. Letterlijke identifiers
(hostnamen, gebruikersnamen, netwerkdetails) zijn vervangen door generieke
placeholders — de instructies zijn daardoor ook herbruikbaar voor wie zelf
zoiets bouwt.
